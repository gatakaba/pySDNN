class pysdnn.coding.PatternCoding(binary_vector_dim, division_num, reversal_num=1)[source]¶

Bases: object

PatternCodingクラスは実数とコードパターンの対応関係を管理する

コードパターンとは{-1,1}を要素とする多次元ベクトルであり,次のような射影関係がある

パターンは以下の条件を満たす

  • 重複が無い
  • -1と1の数の数が等しい
  • 入力次元ごとに異なるパターンを持つ
