class pysdnn.coding.PatternCoding(code_pattern_dim, input_division_num, reversal_num, input_dim)[source]¶

Bases: object

PatternCodingクラスは実数とパターンコードの対応関係の管理を行うクラスです.

パターンコードとは \(\{-1,1\}\) を要素とする \(M\) 次元ベクトルです.

PatternCodingクラスでは \(N\) 次元の実数ベクトルを一括して扱い, コーディングの出力結果は \(N \ast M\) 次元ベクトルです.

Parameters:

パターンコードベクトルの次元数 n

input_division_num : int

実数の分割数 q

reversal_num : int

反転数 r

input_dim : int

入力データの次元数 N

Methods

coding(X, low=0, high=1)[source]¶

コーディングとは実数をパターンコードに変換する操作です.

入力値の値域を引数low,highによって設定することができます.

入力値がlowよりも小さい場合,入力値がlowとした場合のパターンコードを出力します.

入力値がhighよりも大きい場合,入力値がhighとした場合のパターンコードを出力します.

class pysdnn.coding.SelectiveDesensitization(binary_vector_dim, division_num, reversal_num, input_dim)[source]¶

Bases: pysdnn.coding.PatternCoding

SelectiveDesensitizationクラスは実数と選択的不感化されたパターンコードの対応関係を管理するクラスです.

選択的不感化とは二つのパターンコード \(\mathbf{p}_{i}\) , \(\mathbf{p}_{j}\) を以下の式に従って変換を行う操作です.

\[ \begin{align}\begin{aligned}\mathbf{p}_{i,j} = \frac{\mathbf{p}_{i} + 1}{2} \mathbf{p}_{j}\\\mathbf{p}_{j,i} = \frac{\mathbf{p}_{j} + 1}{2} \mathbf{p}_{i}\end{aligned}\end{align} \]

入力次元がN次元の場合, \(N(N-1)\) 個の選択的不感化されたパターンコードが生成されます.

Parameters:

code_pattern_dim : int

パターンコードベクトルの次元数 n

input_division_num : int

実数の分割数 q

reversal_num : int

反転数 r

input_dim : int

入力データの次元数 N
